Position property of css

CSS Position Types
1. static (default)
   - Normal document flow.
   - top, right, bottom, left, z-index do not work.

2. relative
   - Positioned relative to its own original position.
   - top, right, bottom, left, z-index work.
   - Space is still reserved in the layout.

3. absolute
   - Positioned relative to the nearest positioned (non-static) ancestor.
   - Removed from normal document flow.
   - If no positioned ancestor exists, it uses the viewport/body.

4. fixed
   - Positioned relative to the browser viewport.
   - Stays in the same place while scrolling.
   - Removed from normal document flow.

5. sticky
   - Acts like relative until a scroll threshold is reached.
   - Then behaves like fixed within its parent container.
   - Used for sticky headers and navigation bars.

Key Point:
static → default
relative → moves from its own position
absolute → nearest non-static parent
fixed → viewport/browser
sticky → changes with scrolling

FLEXBOX (Flexible Box Layout)

Definition:
Flexbox is a one-dimensional CSS layout model used to arrange items in rows or columns.

Syntax:
.container{
    display: flex;
}

Basic Terms:
• Flex Container → Parent element with display:flex.
• Flex Item → Direct child of flex container.

Axes:
• Main Axis → Direction in which items are placed. horizontal
• Cross Axis → Perpendicular to main axis.  vertical

flex-direction:
• row (default)  l to r
• row-reverse     r to l
• column    top to bottom
• column-reverse   bottom to top

 the direction in flexdirection(verticsl/horizontal) is the directoin of main axis and its perpendicular is cross axis.....(not necesaary that main axis is always horizontal one)
justify content works for main axis and align items for cross axis



Flex Container Properties:
1. justify-content
   Aligns items along the main axis.
   Values:
   • flex-start
   • flex-end
   • center
   • space-between
   • space-around
   • space-evenly

2. align-items
   Aligns items along the cross axis.
   Values:
   • flex-start
   • center
   • flex-end
   • stretch

3. flex-wrap
   Controls wrapping of items.
   Values:
   • nowrap
   • wrap
   • wrap-reverse

4. align-content : centre/flex-start/flex-end  
   Controls spacing between wrapped rows/columns.


5. flex-flow: flex_direction flex-wrap     here we can give 2 properties at once

6. gap:30px      gap between each div        
7. can further give row-gap and column gap    ....

Flex Item Properties:


1. order: 5;   then it will be 5th iv no matter the html order    default order=0

1. align-self : flex-start/flex-end/centre on cross axis
   Aligns an individual item.       the priority will be there for align-self over align-content

2. flex-grow
   Defines how much an item grows. or how much space it will take by default each have 1 flex-grow.....if any class have it 2 then it will take double space than other

3. flex-shrink
   Defines how much an item shrinks.  opp to flex-grow   but not works when there is wrap used 

4. flex-basis
   Sets initial size of an item.

5. flex
   Shortcut for grow, shrink and basis.

Important:
• Row Direction:
  justify-content → Horizontal Alignment
  align-items → Vertical Alignment

• Column Direction:
  justify-content → Vertical Alignment
  align-items → Horizontal Alignment

Advantages:
• Easy alignment of elements.
• Responsive layouts.
• Easy horizontal and vertical centering.
• Less code than older layout methods.

Flexbox vs Grid:
• Flexbox → One-dimensional layout.
• Grid → Two-dimensional layout.

Most Common Code:
.container{
    display:flex;
    justify-content:center;
    align-items:center;
    gap:20px;
    flex-wrap:wrap;
}

TRANSITION Property

=========================
CSS TRANSITIONS
=========================

• Used to smoothly change CSS properties.

Syntax:
transition: property duration timing-function delay;

Properties:
1. transition-property → Property to animate (width, color, etc.) or if we want to tansiltion on all prop just write all
2. transition-duration → Time taken (2s, 500ms)
3. transition-timing-function →
   - ease (default)
   - linear
   - ease-in
   - ease-out
   - ease-in-out
   - steps()    can give no of steps in which transition take place in bracket

4. transition-delay → Delay before animation starts.

Example:
.box{
    transition: width 2s ease;
}

.box:hover{
    width:200px;
}

Important:
• Usually used with :hover.
• Only works on animatable properties.
• 'all' animates all changing properties.

Shorthand Order:
Property → Duration → TimingFunction → Delay    transition: all 5s steps(3) 2s;



----------------------------
TRANSFORM
----------------------------

• Used to change an element's position, size, or rotation without affecting the document layout.

Syntax:
transform: value;

Transform Functions:

1. translate(x, y)
   → Moves an element.
   Example:
   transform: translate(50px, 20px);  if translate(-20px) then move in diagonal towards left   and can do translateX( )
  
2. rotate(angle)                         can also rotate for  3d objects like    transform: rotateX/rotateY/rotateZ(45deg)
   → Rotates an element.
   Example:
   transform: rotate(45deg);

3. scale(x, y)
   → Increases or decreases size.
   Example:
   transform: scale(1.5);    it means increased by 1.5x  on both x and y axis
   scale(2,1);      here, 2 times increase along x axis and 1 times(same) along y axis

4. skew(x-angle, y-angle)
   → Tilts an element.
   Example:
   transform: skew(20deg);


Multiple Transforms:
transform: translate(50px) rotate(45deg) scale(1.2);


-------------------------------------------
.box{
    transition: 0.5s;
}

.box:hover{
    transform: scale(1.2) rotate(10deg);
}
----------------------------------------------

translate() → Move
rotate()    → Rotate
scale()     → Resize
skew()      → Tilt


--------------------------------------
ANIMATION
--------------------------------------

• Used to create continuous or custom animations.

Syntax:

@keyframes animationName{
    from{ }
    to{ }
}



1. animation-name
   → Name of @keyframes.

2. animation-duration
   → Time taken for one cycle.
   Example: 2s

3. animation-timing-function
   → Speed of animation.
   Values:
   ease
   linear
   ease-in
   ease-out
   ease-in-out

4. animation-delay
   → Delay before animation starts.

5. animation-iteration-count
   → Number of repetitions.
   Example:
   2
   infinite

6. animation-direction
   Values:
   normal
   reverse
   alternate
   alternate-reverse

------------------------------------

Shorthand

animation: name duration timing-function delay iteration-count direction;

---------------------------------------------------

@keyframes move{
    from{
        left:0;
    }
    to{
        left:200px;
    }
}


@keyframes animationName{
    0%{}    from
    50%{}   mid                                     we can add multiple mid
    100%{}   to
}

.box{
    position:relative;
    animation: move 2s linear infinite;
}

--------------------------------------------------------


Difference

Transition:
• Triggered by an event (e.g., :hover)
• Two states only

Animation:
• Runs automatically
• Can have multiple keyframes
• Can repeat infinitely

------------------------------------

Interview Points

• @keyframes defines animation.
• animation-name links element to keyframes.
• animation-duration is mandatory.
• animation-iteration-count: infinite → runs forever.
• animation-direction controls animation direction.
